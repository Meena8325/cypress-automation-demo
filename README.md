# Cypress Automation Demo

A test automation framework built using **Cypress** and **JavaScript**, demonstrating end-to-end testing on the [SauceDemo](https://www.saucedemo.com/) web application.

## 🛠 Tech Stack
- Cypress
- JavaScript
- Node.js

## 📁 Project Structure
cypress/
e2e/
SauceDemo/
Login.cy.js
fixtures/
support/

## ✅ Test Coverage
- Login functionality with valid credentials
- URL/dashboard verification after login

## 🚀 How to Run

1. Clone the repository
```bash
git clone https://github.com/Meena8325/cypress-automation-demo.git
```

2. Install dependencies
```bash
npm install
```

3. Open Cypress Test Runner
```bash
npx cypress open
```

4. Select **E2E Testing** → choose a browser → run the test
